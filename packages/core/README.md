# persona-core

Open-source Python library for building AI personas with typed memory.

## Install

```bash
pip install persona-core
```

For local GPU inference:
```bash
pip install persona-core[local]
```

For Postgres-backed memory stores:
```bash
pip install persona-core[postgres]
```

## Quick start

```bash
persona init
persona chat persona.yaml
```
